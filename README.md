# Event Management System

A full-stack **Event Management System** built using **Flask (Python)** for backend, **HTML, CSS, JavaScript** for frontend, and SQLite for database management. This project helps users create and manage events, handle inventory allocation, and track availability and pricing efficiently.

---

## Features

### Authentication System
- Secure login/logout using Flask sessions
- Session-based access control
- Protected dashboard routes

### Event Management
- Create new events with details like name, date, time, and phone number
- Edit and update existing events
- Delete unwanted events
- View upcoming and past events

### Inventory Management
- Add and manage inventory items
- Track total quantity and available stock
- Allocate inventory to specific events
- Prevent over-allocation beyond available stock

### Pricing & Calculation
- Automatic calculation of total cost
- Item-wise price × quantity calculation
- Event-level cost summary

### Dashboard
- Overview of events
- Inventory summary
- Quick navigation between modules

### Payments 
- Gives details of all the events where the invntory is allocated 
- gives generate bill option to dwonload the pdf of the bill 

### orders
- Gives read-only view of all the details of the events 

### Responsive UI
- Mobile-friendly design
- Adaptive layouts using CSS media queries
- Clean table and card-based UI
